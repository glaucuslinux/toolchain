glaucus's x86-64 toolchain

## cerata
* `binutils`: 2.35
* `gcc`: 10.2.0
* `gmp`: 6.2.0
* `isl`: 0.22.1
* `libgomp`: 10.2.0
* `libstdc++-v3`: 10.2.0
* `mpc`: 1.2.0
* `mpfr`: 4.1.0
* `musl`: 1.2.1

## Features
* Graphite loop optimizations: `isl`
* Open Multi-Processing (OpenMP): `libgomp`
* Supported LTO compression algorithms: `zstd` (default) and `zlib` (fall-back)
